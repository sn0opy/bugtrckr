# bugtrckr

This will be a simple bug tracking system. When it's done ...

Features:

* Ticketsystem
* Milestones
* Projects
* Timeline
* Users
* Localization

## Requirements
* PHP 5.3
* PHP Intl (recommended)
* SQLite
* PDO
* PDO SQLite
